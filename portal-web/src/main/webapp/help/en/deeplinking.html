The maps within the data portal are deep linkable using the provided HTML in the "Embed" text box below the map.

See the species page Puma concolor for an example.

Screenshot

They are available in 3 sizes (width by height):

  • Small: 360px by 226px
  • Medium: 548px by 320px
  • Full: 730px by 410px


The default within the "Embed" text box is 730px by 410px.
This can be adjusted by appending "?size=small" or "?size=medium" to the src attribute,
and adjusting the width and height attributes to the values listed above. e.g.
